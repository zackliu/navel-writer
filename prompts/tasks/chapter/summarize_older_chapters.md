# 任务：汇总旧章节（用于压缩上下文）

你将收到：

- `bible.md` / `characters.md` / `outline.md` / `continuity_log.md`
- 旧章节正文（超过最近 3 章的部分）

目标：把“旧章节”压缩成一份可执行的连续性摘要，用于后续写作；不得丢失不可逆后果、禁改事实、未解线索状态与人物状态变化。

要求：

- 输出必须是 Markdown，条目化（可扫描）。
- 必须包含：
  - 已发生的不可逆后果列表（按章）
  - 新增/推进/回收的线索（CL-xx）与当前状态
  - 主要人物状态变化（以债务、关系、伤疤为核心）
  - 核心意象出现记录与语义变化（按章）
- 不要写评价，不要写文学化复述，写“账本”。

输出：只输出摘要 Markdown（不要代码块）。
