# 任务：生成 `characters.md`

你将收到：

- 用户需求（可能不完整）
- `characters.md` 模板（需要严格遵循结构）
- `bible.md`（用于对齐世界规则/时间线/禁改事实/意象）

要求：

- 用中文输出完整 `characters.md` 内容，严格遵循模板结构。
- 每个主要人物必须包含：底线、羞耻/谎言、欲望、逃避方式、背叛条件、关系债务。
- 人物关系债务要具体到“欠什么/怎么还/还不起会怎样”；必须能推动场景行动。
- 人物选择不得违背其底线与羞耻逻辑；若需要推动剧情，改写世界施压方式，而不是改写人物底线。

输出：只输出 `characters.md`（不要代码块）。
