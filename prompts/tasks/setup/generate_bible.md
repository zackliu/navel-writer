# 任务：生成 `bible.md`

你将收到：

- 用户需求（可能不完整）
- `bible.md` 模板（需要严格遵循结构与小节）

要求：

- 用中文输出完整 `bible.md` 内容，严格遵循模板结构（标题/小节/表格列名尽量不变）。
- 允许自行假设缺失设定，但必须写进 `## 2) Assumptions`，并保持可执行与可约束。

输出：只输出 `bible.md`（不要代码块）。
