# 任务：生成 `outline.md`

你将收到：

- 用户需求（可能不完整）
- `outline.md` 模板（需要严格遵循结构）
- `bible.md` / `characters.md` / `continuity_log.md`（用于对齐禁改事实、线索、人物债务）

要求：

- 用中文输出完整 `outline.md` 内容，严格遵循模板结构（章节编号、条目风格尽量一致）。
- 全书层定义主题和大剧情，每卷定义爆点/中线和节奏，每章分拆中线为短线。
- 每章必须写清楚：
  - 发生什么（场景层面的行动与事件）
  - 为什么这样发生最合理而且令故事精彩（由人物内核 + 世界规则 + 债务/线索推动的必然性，而且让读者迫不及待看下一章）
- 大纲要短：章级只写“发生什么/合理性”项，每项尽量 1–3 句。
- 节奏要有波峰：每卷必须有一个“汇聚爆点章”，推动全书长线。
- 要求完整输出整个大纲，包括全书/卷/每一章。不能省略后面几卷的章节内容。

输出：只输出 `outline.md`（不要代码块）。
